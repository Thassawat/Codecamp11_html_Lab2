การหาข้อมูล Tags

  1. <!Doctype> มีหน้าที่อะไร
  2. ถ้าอยากให้มี tooltip ใน Tag นั้น ต้องใช้ Attribute อะไร
  3. Tag <b>, <i> ต่างจาก <strong>, <em> หรือไม่ อย่างไร
  4. หา Tag ที่แสดงข้อความกลับหลังได้ โปรแกรมเมอร์ แสดงเป็น โปรแกรมเมอร์
  1. <!Doctype> เป็นข้อมูลแจ้งbrowserถึงมาตรฐานที่จะมีการใช้ในเอกสารเว็บเพจนั้นๆเป็นไปตามมาตรฐานใด เช่น <!Doctype html> แสดงว่าเอกสารเว็บเพจนี้จะเป็นไปตามมาตรฐานhtml
  2. tooltip ใช้ Attribute <title>
  3. tag<b> เป็นการทำตัวหน้าให้กับข้อความในtagนั้นๆ ในขณะที่<strong>เป็นการเน้นถึงความสำคัญของข้อความในtag
    tag<i>เป็นการทำตัวเอียงให้กับข้อความในtagนั้นๆ ในขณะที่<em>เป็นการเน้นย้ำข้อความในtagที่ต้องการ(จะมีการเน้นเสียงเมื่อใช้screen reader)
  4. ใช้tag<bdo dir="rtl">โปรแกรมเมอร์</bdo>เพื่อให้แสดงเป็น โปรแกมเมอร์
